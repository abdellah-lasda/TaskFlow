# 🚀 TaskFlow

A scalable full-stack task management application designed to help users organize, track, and manage daily workflows efficiently.

## 🛠 Tech Stack

**Frontend:** React, Redux Toolkit, Tailwind CSS  
**Backend:** Node.js, Express.js  
**Database:** MongoDB  
**Authentication:** JWT  

---

## ✨ Features

- 🔐 Secure JWT authentication
- 📝 Create, update, delete tasks
- 📌 Task status management
- 🔎 Filtering & search functionality
- 🛡 Protected routes & middleware
- 📱 Responsive UI design

---

## 📂 Project Structure

```
client/
server/
```

- `client` → React frontend
- `server` → Express API & database logic

---

## 🚀 Installation

### 1️⃣ Clone the repository
```
git clone git@github.com:abdellah-lasda/TaskFlow.git
```

### 2️⃣ Install dependencies

**Backend**
```
cd server
npm install
```

**Frontend**
```
cd client
npm install
```

### 3️⃣ Environment Variables

Create a `.env` file inside `server`:

```
PORT=5000
MONGO_URI=your_mongodb_connection_string
JWT_SECRET=your_secret_key
```

---

## ▶️ Run the Project

**Backend**
```
npm run dev
```

**Frontend**
```
npm start
```
