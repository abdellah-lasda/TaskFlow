# TaskFlow
  TaskFlow – Full-Stack Task Management System
Description:
  A scalable full-stack task management application designed to help users organize, track, and manage daily workflows. Built with React, Node.js, Express, and MongoDB, featuring secure authentication, RESTful APIs, and responsive UI design.
Key Highlights:
  JWT-based authentication system
  Full CRUD operations
  Protected routes & middleware
  REST API architecture
  Clean and modular folder structure
